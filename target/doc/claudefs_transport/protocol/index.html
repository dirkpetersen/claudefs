Protocol definitions and serialization for transport layer

This module provides the binary RPC protocol for ClaudeFS transport. The protocol uses frame-based messaging with a fixed 24-byte header.

NOTE: This is a skeleton module with type definitions for reference. Full implementation will be completed by A4 (Transport) agent.
